### Verified clean ✅

- **Security headers all present on `/`**:
  - `strict-transport-security: max-age=31536000; includeSubDomains; preload`
  - `x-frame-options: DENY`
  - `x-content-type-options: nosniff`
  - `permissions-policy: camera=(), microphone=(), geolocation=()` (plus 14 more)
  - `referrer-policy: strict-origin-when-cross-origin`
- **Quote token entropy**: UUID v4 format (36 chars, 122 bits of entropy). Doc requires ≥20 chars.
- **Rate limit on `/api/onboard`**: 5 attempts returned 400 (validation), 6th returned `429 {"error":"Too many submissions. Please wait a few minutes and try again."}`. PR#42 fix verified.
- **OAuth CSRF protection**: `/api/auth/start/google` sets `oauth_state_google=<random>` cookie with `HttpOnly; Secure; SameSite=Lax; Max-Age=600`, matching the `state=` param in the redirect URL. PR#42 fix verified.
- **OAuth callback rejects forged state**: `?state=mismatched` (no cookie match) returns 307 → `/settings?google_error=invalid_state`. Friendly, audited error path.
- **Cron endpoints guarded**: hitting `/api/cron/quote-chase` from authenticated browser session returns `401 {"error":"Unauthorized"}`. The guard is correctly preventing manual triggering.
- **Failed audit rows display red**: `*_failed` rows verified to receive distinct styling.
